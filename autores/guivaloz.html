CMS de Movimiento Libre

Administrador de contenidos programado con Ruby para crear un sitio web con páginas estáticas.

Autor guivaloz

Descargar el CMS de Movimiento Libre

Descargue la versión más reciente del CMS de Movimiento Libre en GitHub por medio del comando git.

git clone git@github.com:MovimientoLibre/cms.git

O ingrese al repositorio público desde su navegador de internet:

https://github.com/MovimientoLibre/cms

Cualidades de la versión actual:

  • Basado en Twitter Bootstrap.
  • Puede alimentarse de archivos markdown. Busca los que tengan extensión md
  • Compatible con los archivos con extensión rb que contienen objetos de Ruby y contenido en RedCloth.
  • Puede usar rutas relativas y absolutas. Identifica las rutas relativas para ajustar todos los vínculos.
  • Y muchas novedades más…

Por hacer

  • Que la página de inicio elabore la paginación.
  • Que identifique las imágenes y agregue código para escalar con bootstrap.

Markdown: la forma fácil de escribir para web

Este artículo es una traducción de Markdown Syntax Documentation.

Markdown es una forma fácil de escribir y de leer para crear archivos HTML. Un archivo escrito en makdown es de texto plano, es to significa que puede ser elaborado en cualquier editor de texto. Cabe recalcar que mientras el HTML es un formato para publicar en la web, el markdown es un formato para escribir en la web. En ese concpeto, entonces markdown no pretende ser un sustituto del HTML, sino una forma cómoda para escribir contenidos.

Versión anterior

Por el año de 2008 comenzó a escribirse este software. Para fines históricos, puede descargar la versión 1.3 al 16/mayo/2008 si lo desea estudiar.

Requerimientos

Para usar el CMS de Movimiento Libre necesita tener instalado:

  • GNU/Linux
  • Servidor Apache HTTP previamente configurado y probado.
  • Ruby versión 1.9.
  • La gema RedCloth.
  • La gema Kramdown.

Textile

Textile es un lenguaje de marcado ligero originalmente desarrollado por Dean Allen como generador de texto web humano.

Textile convierte el texto marcado como tal en XHTML válido. También convierte algunos caracteres a su código HTML y les hace mejoras; como las comillas dobles a comillas dobles que abren al inicio y comillas dobles que cierran al final.
